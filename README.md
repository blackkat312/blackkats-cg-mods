To download:
1. Click the code menu and press download zip.
2. Unzip the files.
3. Copy everything in the the_sprites folder and paste them into the [your_download]\sprites folder. Replace when prompted.
4. Copy everything in the sprite_dicts folder and paste them into the [your_download]\sprites\dicts folder. Replace when prompted.
5. Copy everything in the kitty_lang folder and paste them into the [your_download]\resources\lang\en\cat folder. Replace when prompted.

If anything doesn't work or is confusing, you have perms to ping me in the thread!
